# Palette's Journal

## 2024-04-04 - Conversational UX insights
**Learning:** This is a WhatsApp bot. Standard web UX improvements (HTML/CSS, ARIA) do not apply. In text-based conversational interfaces, emojis act as primary visual indicators (similar to color or icons in standard GUIs). Hardcoded "positive" emojis for potentially negative states (like a deficit balance) break the conversational UX.
**Action:** Adapt the "Palette" mindset to a text-based, conversational UI by utilizing dynamic emojis to reflect the state correctly.
